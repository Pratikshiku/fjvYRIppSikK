### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 三、功能描述
系统分为（身份）：普通用户、志愿者、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 活动信息
5. 公益信息
6. 留言板
7. 后台管理
8. 个人中心
9. 我的收藏

**志愿者功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 活动报名管理
6. 取消报名管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 志愿者管理
5. 申请志愿者管理
6. 活动信息管理
7. 活动报名管理
8. 取消报名管理
9. 系统管理（关于我们、系统简介、轮播图管理、公益新闻）
